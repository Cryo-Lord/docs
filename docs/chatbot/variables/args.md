# $(args)

Outputs an argument from the command.

#### Parameters

This variable does not take any parameters.

#### Example Input

```
$(1)
```

#### Example Output

```
Hello
```

## $(args.emote)

Outputs an argument only if it is an emote.

#### Parameters

This variable does not take any parameters.

#### Example Input

```
$(1.emote)
```

#### Example Output

```
Kappa
```

## $(args.word)

Outputs an argument only if it does not contain symbols.

#### Parameters

This variable does not take any parameters.

#### Example Input

```
$(1.word)
```

#### Example Output

```
Hello
```
