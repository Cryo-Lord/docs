# $(ffzemotes)

Lists the active FrankerFaceZ emotes in the channel.

#### Parameters

This variable does not take any parameters.

#### Example Input

```
$(ffzemotes)
```

#### Example Output

```
LULW KEKW 5Head PogU
```
