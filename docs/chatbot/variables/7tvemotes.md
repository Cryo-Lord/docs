# $(7tvemotes)

Lists active 7TV emotes in the channel.

#### Parameters

This variable does not take any parameters.

#### Example Input

```
$(7tvemotes)
```

#### Example Output

```
LULW KEKW 5Head DinkDonk PogU
```
