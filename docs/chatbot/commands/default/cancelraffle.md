# !cancelraffle

Cancels an ongoing raffle.

### Module

- The `!cancelraffle` command is part of the [raffle module](../../modules/raffle).

### Arguments

- None

### Example Input

```
!cancelraffle
```

### Example Output

```
The raffle was canceled! 
```
