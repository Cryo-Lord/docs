# !enablesfx

The `!enablesfx` command is used to enable all sound effect items in the loyalty stream store that were previously disabled.

### Arguments

- None

#### Example Input

```
!enablesfx
```

#### Example Output

```
@Styler, successfully updated 1 items. 
```
