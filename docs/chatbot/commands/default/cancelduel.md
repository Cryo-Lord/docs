# !cancelduel

Cancels an ongoing duel.

### Module

- The `!cancelduel` command is part of the [duel module](../../modules/duel).

### Arguments

- None

### Example Input

```
!cancelduel
```

### Example Output

```
@styler, darkoe denied your duel :( 
```
