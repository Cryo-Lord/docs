# !hypecup

The `!hypecup` command allows for clearing the hypecup overlay.

#### Arguments

- `clear` - Clears the hypecup overlay.

#### Example Input

```
!hypecup clear
```

#### Example Output

```
@Adeithe, Successfully cleared HypeCup.
```
